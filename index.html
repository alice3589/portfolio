自己紹介

三重県を拠点に、専門学校で情報系を学んでいます。卒業研究では Java 17 / Maven を使った 「火星のテラフォーミング」シミュレーションに取り組み中です。

ゲームのスピードランが好きで、その延長で AutoSplitter（自動計測）や速度向上 Mod、 走行分析ツールを自作しています。学習用に Java の入門サイトを公開するなど、 「学んだことを形に残す」ことを大切にしています。

  • 計測・解析・自動化ツールの設計
